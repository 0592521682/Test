What is CI/CD?

CI/CD is a way to keep code moving smoothly from my computer to the live site. Instead of me copying files and hoping nothing breaks, a workflow runs tests and builds the site every time I push. That means fewer mistakes and more time for learning.

My Pipeline

When I push changes to the `main` branch, GitHub Actions picks them up. First it makes sure `index.html`, `style.css`, and `script.js` exist, then it uploads everything and deploys the page. In a few minutes the live URL shows the new content without me doing anything else.
